#  WildThoughts

## Instructions
Run the following commands in your terminal:
``` shell
conda create --name wildthoughts python=3.9.18
```
``` shell
conda activate wildthoughts
```
``` shell
pip install -r requirements.txt
```
``` shell
script.py all
```
``` shell
manage.py runserver
```
It will take some time, so please be patient.
